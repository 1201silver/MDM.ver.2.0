#### Git Commit Message Style Guide
Jsp 🌸 Spring ⚙️
- `✨feat` : 새로운 기능 추가
- `🐛fix` : 버그 수정
- `📝docs` : 문서 내용 변경
- `💄style` : CSS 등 사용자 UI 디자인 변경
- `✅test` : 테스트 코드 작성
- `♻️refactor` : 코드 리팩토링
- `📦chore` : 환경설정 및 프로젝트 세팅
- `🚀deploy` : 배포
- `🔥remove` : 파일이나 코드 삭제
